# HRnet

Application web de gestion des employés développée avec React et Vite, permettant de visualiser et gérer les données des employés de manière efficace.

## Comment le démarrer

Ce projet est une application React moderne. Pour le lancer, il est recommandé d'utiliser les commandes suivantes :

### Prérequis

-   Node.js (version 16 ou supérieure)
-   npm ou yarn

### Installation

1. Clonez le dépôt ou téléchargez les fichiers du projet
2. Installez les dépendances :
    ```bash
    npm install
    ```

### Lancement en développement

Pour lancer le serveur de développement :

```bash
npm run dev
```

Cela lancera un serveur local (généralement sur http://localhost:5173)
